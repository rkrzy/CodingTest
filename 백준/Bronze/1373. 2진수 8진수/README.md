# [Bronze I] 2진수 8진수 - 1373 (다른 사람의 코드를 참조함)

### 고민한 부분
1. 문자열을 1,000,000자리까지 입력을 받을 수 있다는 부분에서 오류를 많이 범했다.
2. 2진수를 8진수로 변경하는 부분에서 오류가 많이 발생했다.
3. 1번 부분을 해결할 수 없어서 처음에 Scanner로 수행했던 부분을 BufferedReade로 변경해서 해결을 했다.
