# Stack(스택)관련 내용 정리

Stack : LIFO구조를 가지고 있는 자료주고 Restricted Structure로 불리기도 한다.

### 스택의 성질
- 원소의 추가가 O(1)
- 원소의 제거가 O(1)
- 제일 상단의 원소 확인이 O(1)
- 제일 상단이 아닌 나머지 원소들의 확인/변경이 원칙적으로 불가능

### 구현
- 배열, 연결리스트 구현 가능
